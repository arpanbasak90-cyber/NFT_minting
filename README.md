## ⚙️ How It Works

### 🎨 Mint NFT

* User inputs Token ID + Metadata
* Contract stores NFT on-chain
* Ownership assigned securely

### 🔄 Transfer NFT

* Owner initiates transfer
* Contract validates ownership
* Updates ownership
